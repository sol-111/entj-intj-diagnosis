# 恋愛タイプ診断：ENTJ男 × INTJ女

> 使用データ：MBTI / エニアグラム / ビッグファイブ / ソシオニクス
> 分析フレームワーク：6つの愛のスタイル（John Lee）/ 5つの愛情言語 / アタッチメント理論

---

## フレームワーク①：6つの愛のスタイル（Love Styles）

| スタイル | 意味 | 特徴 |
|---------|------|------|
| Eros | 情熱型 | 強烈な惹かれ、理想のパートナー像を持つ |
| Storge | 友情型 | 友人関係が恋愛に発展。ゆっくり深まる |
| Pragma | 実用型 | 相性・将来性・条件で合理的に判断 |
| Agape | 献身型 | 見返りを求めず相手のために尽くす |
| Ludus | 遊戯型 | 恋愛をゲームとして楽しむ。束縛嫌い |
| Mania | 執着型 | 強烈な愛着・不安・嫉妬が混ざる |

## ENTJ男（shoki）の恋愛タイプ

### 主スタイル：Pragma（実用型）+ Eros（情熱型）

**推定根拠**

| 特徴 | 元データ | 意味 |
|------|----------|------|
| 目標・将来設計重視 | ENTJ + タイプ1（改革者） | 恋愛も「意味のある関係か」で判断する |
| 高い基準を持つ | タイプ1の完璧主義 | パートナーへの期待水準が高い |
| 行動で愛を示す | ENTJ実行力 + 協調性34%（低） | 感情表現より「してあげること」で愛を示す |
| 情熱と信念 | タイプ7ウィング + Eros要素 | 惹かれたら真っ直ぐに追う |
| 感情より論理 | ビッグファイブ情動性が低い | 感情的なドラマより安定した関係を好む |

**恋愛における行動パターン**
- 相手の知性・価値観・目標を最初に見る（直感的に「この人は本質的か」を判断）
- 感情の起伏は少なく見えるが、決めた相手へのコミットメントは強い
- 口より行動。デートの計画・問題の解決・将来の設計で愛情を示す
- パートナーにも「成長・向上心」を求める。だれでもいい関係は求めない
- タイプ7ウィングにより、退屈を嫌う。刺激のある関係・新しい体験を共有したい

**恋愛の弱点**
- 批判的な指摘が愛情の表れでも、相手には「責められている」と映りやすい
- 「察してほしい」という相手の感情ニーズに気づきにくい
- ストレス時はタイプ4方向（内向的・自己批判）に向かい、急に引きこもる

## INTJ女（ゆう）の恋愛タイプ

### 主スタイル：Storge（友情型）+ Mania（執着型）混合

**推定根拠**

| 特徴 | 元データ | 意味 |
|------|----------|------|
| 信頼関係から始まる | INTJ内向性 + タイプ4の慎重さ | 急に心を開かない。時間をかけて深まる |
| 強烈な感情の内側 | 情動性73%（高） | 外から見えないだけで感情は激しい |
| 自己価値への不安 | INTJ-T（乱気流型）+ タイプ4 | 「本当に愛されているか」を繰り返し確認したくなる |
| 深い一体感を求める | タイプ4（個人主義者）の欲求 | 「この人だけに理解される」という唯一性を求める |
| 独立しながらも深く | タイプ5ウィング + 協調性16% | 依存はしないが、精神的な深さは強く求める |

**恋愛における行動パターン**
- 最初は壁が高く、なかなか心を開かない。心を開くまでに時間がかかる
- 一度信頼したら非常に深く・誠実に関わる
- 感情の波は外に出さず、内側で処理しようとする（でも実際は激しい）
- 「自分だけに特別な存在として扱われたい」という欲求が強い
- タイプ9のスコアも高いため、衝突を避けて内側に溜め込みやすい
- タイプ3スコアが高い → パートナーの前では「デキる自分」を見せたい気持ちも

**恋愛の弱点**
- 感情を言語化するのが苦手。「わかってほしいが伝えられない」になりやすい
- 欠乏感（タイプ4のコア）から、無意識に試し行動や感情的引き篭もりが出ることも
- INTJ-Tの自己批判が強く、恋愛でも「自分には価値がないかも」と揺れることがある
- 外向性が低い（20%）ため、相手が社交的な場面で消耗しやすい

## フレームワーク②：5つの愛情言語（Love Languages）

> 愛情の「伝え方」と「受け取り方」の分析

| 愛情言語 | 意味 |
|---------|------|
| 肯定の言葉 | 言葉で直接伝える・褒める・認める |
| 質の高い時間 | 二人だけの集中した時間を共有する |
| 贈り物 | 物や記念日など形ある愛情表現 |
| 奉仕 | 行動・サポートで相手を助ける |
| スキンシップ | 身体的な触れ合い・近さ |

### ENTJ男（shoki）の愛情言語

**与え方：奉仕（Acts of Service）**
- ENTJ的に「行動すること」が愛情の示し方
- 問題を解決する、計画を立てる、相手の目標を支援する

**求め方：肯定の言葉（Words of Affirmation）**
- タイプ1の根底には「自分は正しいか」という問いがある
- パートナーからの承認・肯定の言葉が心の安定につながる

### INTJ女（ゆう）の愛情言語

**与え方：質の高い時間（Quality Time）**
- 外向性が低く社交が苦手なINTJにとって、誰かと時間を共有すること自体が大きな愛情表現
- 「あなただけと過ごす時間を選んだ」というメッセージを込める

**求め方：肯定の言葉（Words of Affirmation）**
- タイプ4の欠乏感と自己批判（INTJ-T）が相まって、「あなたは特別だ」「理解している」という言葉を深く求める

### 愛情言語の噛み合いチェック

| 項目 | 評価 |
|------|------|
| 男が与える「奉仕」× 女が求める「言葉」 | ❌ ズレ：行動してもらっても「言葉で言ってほしい」が満たされない |
| 女が与える「時間」× 男が求める「言葉」 | △ 部分一致：一緒にいる時間は共有されるが、肯定の言葉が不足しがち |
| 両者の「求め方」は同じ（言葉） | ✅ 一致：互いに言語化・承認の言葉を意識すれば解決できる |

> **改善策**：男は意識して「言葉で気持ちを伝える」練習を。女は行動への感謝を言葉で返すと男も満足しやすい。

## フレームワーク③：アタッチメントスタイル（愛着スタイル）

| スタイル | 特徴 |
|---------|------|
| 安定型 | 信頼感があり、距離感が適切。感情も安定 |
| 不安型 | 見捨てられ不安が強く、愛情確認を求めがち |
| 回避型 | 近づきすぎると引いてしまう。自立を優先 |
| 恐怖回避型 | 近づきたいが怖い。矛盾した行動が出る |

### ENTJ男（shoki）：**回避型（Avoidant）寄り**

- ビッグファイブ情動性が低い → 感情的な絆の必要性をあまり感じない
- 協調性34%（低）+ タイプ6が最低 → 依存・束縛を嫌う
- ストレス時にタイプ4方向（引きこもり）に動く → 急に連絡が減ることも

> ただしENTJ-A（アサーティブ）なので、健全な状態では「安定型」に近い。

### INTJ女（ゆう）：**不安型（Anxious）寄り**

- 情動性73%（高）+ INTJ-T + タイプ4 → 感情的な波があり、見捨てられ不安が出やすい
- タイプ9高スコア → 衝突を避けるために感情を抑圧し、内側で不安が膨らむ
- 外向性20%（低）→ 社会的なネットワークが限られ、パートナーへの精神的依存が高まりやすい

### アタッチメントの組み合わせリスク

**「回避型 × 不安型」は最もよく見られるカップルのパターンであり、同時に最も衝突しやすい**

| シチュエーション | 男の反応 | 女の反応 | 結果 |
|----------------|---------|---------|------|
| 男が引いたとき | 「一人の時間が必要」 | 「嫌われた？」と不安が爆発 | 女がさらに追う → 男がさらに引く |
| 女が不安になったとき | 「なぜそんなに気にするの？」と合理的に対応 | 「わかってもらえない」と傷つく | 溝が広がる |

**このパターンを抜け出すために**
- 男：「少し時間をくれ」と事前に伝える習慣を作る（突然の沈黙が最も危険）
- 女：不安を感じたら内側で処理せず、言葉にして伝える練習をする

## 恋愛タイプ総合サマリー

| 項目 | ENTJ男 | INTJ女 |
|------|--------|--------|
| 恋愛スタイル | Pragma + Eros（実用的で情熱的） | Storge + Mania（友情から深まり、感情激しい） |
| 愛情の与え方 | 行動・奉仕・計画 | 時間・誠実さ・深い関与 |
| 愛情の求め方 | 肯定・承認の言葉 | 特別扱い・理解の言葉 |
| アタッチメント | 回避型寄り | 不安型寄り |
| 恋愛の核心 | 「共に成長できるか」 | 「唯一理解してもらえるか」 |

### この2人が上手くいくカギ

1. **男は「言葉」を使う**：行動だけでなく、「好き」「一緒にいたい」「信頼している」を声に出す
2. **女は「不安を言葉にする」**：溜め込まず、早い段階でシンプルに伝える（責めない形で）
3. **「引く」前に一言**：男が一人の時間を必要とするとき、事前に「少し時間がほしい」と伝えるだけで女の不安は大きく減る
4. **深い対話の時間を作る**：どちらも表面的な関係は求めていない。定期的に「本音で話す時間」を設計する

> **一言：「愛し方が違うだけで、求めているものの深さは同じ。言語化の習慣が2人の関係を変える」**
