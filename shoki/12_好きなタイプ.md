## 認知機能が決める「好き」の構造

### 認知機能スタック

Te（外向思考）→ Ni（内向直観）→ Se（外向感覚）→ Fi（内向感情）

この順番が「何に惹かれるか」を決定的に規定している。

### Te主導：「行動で手に入れたい」

Te（外向思考）が1番 = 効率・成果・結果で世界を処理する。恋愛でも **「自分が動いて、計画して、手に入れた」** 実感がないと燃えない。

- 店を調べる、デートを計画する、LINEも自分から——**「してあげる」で充実感を得る**
- 最初から相手がグイグイ来ると「価値が下がる」と無意識に感じる
- 恋愛における **Pragma（実用型）** の根拠 = 相手の知性・価値観・将来性をTeで査定している

> **惹かれるタイプ：** 簡単に手に入らない人。追いかけさせてくれる人。でも脈がゼロじゃない絶妙なライン。

### Ni補助：「未来を共有できるか」

Ni（内向直観）が2番 = 長期ビジョン・本質の把握・未来予測。恋愛でも **「この人との未来が見えるか」** を無意識にスキャンしている。

- 「この人と一緒にいて、5年後どうなってるか」を直感で感じ取る
- 将来の話ができない相手、方向性が合わない相手には心が動かない
- MC射手座 = 「何のために生きるか」のベクトルが同じ人に惹かれる

> **惹かれるタイプ：** 自分も確固たる軸と目標を持っている人。「一緒に何かを成し遂げられる」感覚がある人。

### Se第三：「存在感に反応する」

Se（外向感覚）が3番 = 今この瞬間・物理的存在・身体性。恋愛では **「見た目」「雰囲気」「存在感」** にも確実に反応する。

- SLEのSe主導とも連動 → 外見に無関心ではいられない
- ただし「かわいい」より「きれい」「かっこいい」が刺さる
- 知性が滲む美しさ、芯の強さが見える佇まいに惹かれる

> **惹かれるタイプ：** きれい系。シャープな輪郭。華やかすぎず、洗練された美しさ。

### Fi劣等：「自分だけの特別」を求める

Fi（内向感情）が4番 = 最も未発達で最も制御できない。**これがENTJの恋愛における最大の急所。**

- 感情を大量に浴びせられると処理不能 → **感情的なドラマは苦手**
- でもFiは内側にある → **「静かだけど深い感情」にはめちゃくちゃ惹かれる**
- 「自分にだけ見せてくれる」特別感 = **Fiが最も渇望するもの**
- 月さそり座8H = 深い親密さ・秘密の共有の中に安らぎを求める

> **惹かれるタイプ：** 表面はクール。でも自分にだけ本音を見せてくれる。感情の出し方が静かで、奥が深い人。

### SLEのNi示唆機能：無意識に求めるもの

ソシオニクスSLEの5番機能 = **Ni（内向直観）**。最も求めているが、自力では満たせない。

**長期ビジョン・未来予測・物事の深い意味を読む力** を持っている相手に、無意識レベルで惹かれる。Ni主導型（ILI / INTJなど）と一緒にいると「なぜか楽」「なぜかしっくりくる」と感じる。

これが「一緒にいると頭が整理される」感覚の正体。相手のNiが、自分のSLEに足りないビジョンを自然に補ってくれている。

### まとめ：認知機能が描く理想像

```

Ni  → 未来を共有できる。同じ方向を向いてる
```

## シャドウ・アニマが決める「運命の引力」

### シャドウとは

自分が「自分ではない」と思い込んでいるが、確実に自分の一部であるもの。意識が拒絶するほど、無意識で強く作動する。恋愛では「自分のシャドウを自然に体現している相手」に最も強く惹かれる。

### ENTJのシャドウ：抑圧された3つの自分

**1. 「感情的な自分」の否定（Fi抑圧）**

情動安定性90%。「自分は感情に左右されない人間だ」がアイデンティティ。でも月さそり座8Hが示す通り、**内側には極めて強い感情がある。** 消えたのではなく、抑圧されているだけ。

> 「本当は傷ついてる。本当は怖い。本当は『大丈夫じゃなくていいよ』と言ってほしい。——でもそれを認めたら、自分じゃなくなる気がする。」

**2. 「弱い自分」の否定（タイプ1→4分裂）**

普段のタイプ1 = 正しくあれ、完璧であれ。ストレス時にタイプ4に落ちる = **急に内向的・自己批判的になる。** このタイプ4化は「シャドウの噴出」。普段は出ない（タイプ4スコア4）からこそ、出たときの衝撃が大きい。

**3. 「優しい仮面」の葛藤（ASC魚座）**

ASC魚座 = 第一印象は柔らかく受容的。でも本体はENTJ/SLE。**「優しい自分は本物なのか仮面なのか」** ——本人にもわからなくなっている。答えは「両方」。

### アニマ：無意識が求める理想像

ユングの「アニマ」 = 男性の無意識にある女性的元型。**自分が抑圧している側面を、恋愛対象に投影する。**

| ENTJの抑圧 | アニマが求めるもの |
|-----------|-------------------|
| Fi（感情の言語化） | 自分が感じられない感情を代わりに感じてくれる存在 |
| 月さそり座（8H） | 深く、激しく、秘密を共有できる女性 |
| 金星乙女座/獅子座 | 華やかさと実直さの両方を持つ人 |
| 太陽7H | パートナーの中に「自分の完成」を見出す |

### シャドウの引力 = 最も強い恋愛力

**「自分が認められない自分」を自然に体現している相手に惹かれる。**

- **感情的で弱い自分** を否定してる → 感情が豊かで脆さを見せられる人に惹かれる
- **甘えたい自分** を否定してる → 自然に甘えてくる人に惹かれる。一人っ子で甘え方のテンプレートがないからこそ、甘え上手な相手に無意識に引かれる
- **衝動的な自分** を否定してる → 自由で型にはまらない人に惹かれる
- **完璧じゃない自分** を否定してる → 欠点を隠さない人に惹かれる

「意外と社会不適合者なとこが好き」「口悪いとこが好き」「おっちょこちょいなとこが好き」——これは全部 **シャドウへの引力**。自分が見せられない「完璧じゃない部分」を、相手が自然に見せてくれるから惹かれる。

### 投影の罠

- アニマ投影が強いと → 相手を **「理想化」** しすぎる → 現実の相手とのギャップに苦しむ
- 「この人が感情を感じてくれるから自分は感じなくていい」→ **Fiの外注** になる
- 投影を超えるには → **自分自身のFiを育てる** こと。相手に依存するのではなく、自分で感じる練習をする

### シャドウの引力を活かすには

相手を「シャドウの代行者」ではなく「シャドウに気づかせてくれる師」として見る。「この人がいるから感じなくていい」ではなく「**この人がいるから感じられるようになった**」——そうなれた時、その恋は本物になる。

## こういう人は無理（NG・地雷）

### 4. 「すごーい」しか言わない人

何を言っても「すごーい」「さすが〜」。表面的な持ち上げ。

**理由：** タイプ1はお世辞を秒で見抜く。NTJは知的対話を求めてる。「すごーい」= 何も考えてないことの証明。尊敬できない相手には心が動かない。

### 5. 「察して」が多い人

言わないけど怒ってる。態度で示してるのに気づかない相手が悪い。

**理由：** SLEのFi役割機能。暗黙の期待に気づけない構造的限界。「察して」は最も消耗するコミュニケーション。言葉にしてもらわないとわからない。

### 6. 最初からグイグイ来る人

出会ってすぐ「好き」「会いたい」「毎日LINE」。

**理由：** Te主導の恋愛原理。「自分で追いかけて手に入れた」実感がないと燃えない。最初からオープンだと「獲得する対象」じゃなくなる。

### 7. 自分の判断にダメ出しする人

「あなたの判断は間違ってた」（断定形）。

**理由：** タイプ1のコア = 「正しくあること」が存在意義。正しさの否定 = 存在否定。批判されること自体ではなく、**断定形での否定** が最大の地雷。問いかけ形（「こう思うけどどう？」）なら受け入れられる。

## 苦手とすること

- 感情的なドラマや感情の爆発（対処法がわからない）
- 「察してほしい」という暗黙の期待
- 束縛・依存・過度な密着
- 停滞した関係、向上心のない相手
- 自分の批判・指摘が相手を傷つけていると気づかされること
- 論理で解決できない感情の問題に直面すること
- 他の男と比較されること

## 地雷を踏まれたときの反応

| 段階 | 行動 |
|------|------|
| 初期 | 無意識に批判・指摘が増える（ストレスが外に漏れ始める） |
| 中期 | 口数が減る。返信が素っ気なくなる（シャットダウン開始） |
| 末期 | 突然引く。連絡が途絶える（タイプ4化）。「一人の時間が必要」を言わずに消える |

**特徴：** 怒りを爆発させることは少ない。代わりに **「静かにスイッチが切れる」**。外から見ると「急にどうした？」だが、内側では蓄積の臨界点を超えている。
